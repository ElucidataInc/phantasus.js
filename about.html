Project phantasus is a fork from project Morpheus created by Joshua Gould.

We extended the parent project with additional functionality from R and Bioconductor repository:
  • Clustering genes with k-means algorithm;
  • Principal Component Analysis and visualisation of the result with plotly;
  • Differential Expression with limma;
  • Loading datasets from Gene Expression Omnibus by identifier.
